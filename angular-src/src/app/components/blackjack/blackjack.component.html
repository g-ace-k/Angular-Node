A quick summary of this mobile app.

This mobile app was created in Android Studio, using a custom framework and openGL for graphics. I decided to go this route because one I never planned on making really large and complex apps that could benefit from frameworks already out there like Unity or LibGDX that would add a lot of unneccessary weight to the app, and two I wanted to see if I could make a native app from scratch. The game is essentially run by going through a general update and present method, and then depending on the 1 of 6 states of the game runs seperate updates and present methods to detect touch presses and updates based on the touchpoint. All artwork and sounds were also created by me, which was probably the hardest part in making this app, as I am not really a graphic or sound artist. With a custom framework only adding everything that is neccessary to run the game, it clocks in to a little under 33 MBs in size, compared to similar blackjack apps that can be as much as 100 MBs.
