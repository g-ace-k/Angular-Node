Before you start


Check out all the source code over at my github here.


Here you can check out all the code such as how the backend and frontend are setup, how the contact form mailer works, how it validates when you are logged in and authorizes certain features and much more. You can even clone the github repository to your machine and run it in your local network using this command 

$ git clone https://github.com/g-ace-k/Angular-Node

Setting up the backend


Install dependancies

In the main folder, install dependancies using

$ npm install

Configure config.database file

In the config/database folder, add a database that it can connect to. You can make a free database over at mLab.

module.exports = {{'{'}}
    database: 'DatabaseLinkHere',//connection point to database
    secret: 'secretHere' //Secret used for JWT authentication
{{'}'}}

Run the backend server

In the main folder, use nodemon to start a backend server on your machine on port 5000

$ nodemon

Setting up the frontend


Install dependancies

In the angular-src folder, install dependancies using

$ npm install

Configure http requests

In the angular-src/src/services/auth.service.ts file, change all http requests to point to

return this.http.post('http://localhost:5000/users/...

Run the app

Using angular-cli run the app using ng serve command

$ ng serve

Visit localhost:4200 to view the working app and database.
