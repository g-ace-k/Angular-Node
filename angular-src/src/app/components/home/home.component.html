A quick summary of this web app.

This web app was created using a MEAN stack and ultimately will serve as a template for a future webapp I plan on making. The goal of this web app was to become familiar with using both frontend and backend frameworks as well as becoming familiar with a NoSQL database as I had experience working with MySQL databases already. The creating users and login page doesn't really provide any additional functionality besides allowing a look at all users in the database as a way to check the database is working as expected. I wouldn't advise on putting in real passwords but they are still stored using bcrypt.



What you will find in this web app.

Backend Functionality

Handle all database requests through REST apis on the backend seperate from the client.

MongoDB

Uses a NoSQL database hosted at mLab. for ease of use and scalability.

Angular

Angular framework and angular-cli for generating components, services, etc and local dev server for easy testing.

OAuth and Passport

This might only be a test web app, but hey why not have secure authorization and authentication.

Bootstrap

For speedy development, responsiveness, and consistency for an easy to view web app that works for all screen sizes.
